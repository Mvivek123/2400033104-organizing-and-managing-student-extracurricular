# CampusConnect - Student Extracurricular Management System

This workspace contains **CampusConnect**, a React application for discovering and managing student clubs and events.
All functionality is implemented in the `frontend/` directory; no backend server is required.

### Getting started

```bash
cd frontend
npm install
npm run dev   # alias for npm start
```

Open your browser at `http://localhost:3000` to use the app. Authentication and data persist in local storage.

The frontend code uses:
- React (Create React App)
- Tailwind CSS
- React Router
- LocalStorage for data

Feel free to explore the code under `src/` – components, pages, and context providers are organized clearly for easy extension.
